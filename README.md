# MasterBorder

> Cross-border trade compliance dashboard — built for the Built with Opus 4.7 hackathon.

One developer. Claude Code + Opus 4.7 Managed Agents. Open-source. MIT licensed.

Small exporters face an impossible choice: spend weeks manually checking tariffs, sanctions, HS codes, and regulations across countries — or pay $50,000+/year for enterprise tools like Thomson Reuters World-Check. MasterBorder makes enterprise-grade compliance available to everyone.

**Status:** Building 🚧

## Stack

- **Backend:** Python 3.12, FastAPI, Anthropic SDK, Managed Agents
- **Frontend:** Next.js 15, React 19, Tailwind CSS + shadcn/ui
- **License:** MIT
